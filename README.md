# Scratchsee-desktop
bu JavaScript ile yapılmış Scratchsee uygulaması
# Nasıl derlenir?
## paketleri yükleme 
Pkateleri yükleyebilmek için node.js lazım node.js arsa bu komutu kullanın:

```cmd
npm install
```

## Derleme
Derlemek için bu komutu kullanın:

```cmd
npm run build-all
```
